Python version 3.7 or better.
The following modules are needed:
requests (pip install requests or pip3 install requests )
pandas (pip install pandas or pip3 install pandas)
streamlit (pip install streamlit or pip3 install streamlit)

To start the application:
1. open command prompt/terminal 
2. change directory to the path where main.py is located
3. type the following: python -m streamlit run main.py 
or for mac
type the following: python3 -m streamlit run main.py 

It should open a browser with the address localhost:8501
minimize the terminal window and continue on the browser

To exit the application:
enter Ctrl+C in the terminal window
